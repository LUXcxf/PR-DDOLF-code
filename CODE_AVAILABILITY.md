# Code Availability Notes

The package includes the following code modules:

```text
stage1/models/baselines/
stage1/models/cnn/
stage1/models/transformer/
stage2/models/baselines/
stage2/models/fno/
```

These scripts cover the Stage-I baseline, CNN, Transformer, Stage-II MLP
residual baseline, and Stage-II FNO residual-field correction components.

The scripts are organized to run from the repository root with processed data
and split files placed under the project-relative paths described in
`RUN_CONFIGS.md`.
